# **eCommerce Application (MEAN Stack + Ionic)**

This is a full-stack **eCommerce application** built with the **MEAN stack** (MongoDB, Express, Angular, Node.js) for the web and **Ionic** for the mobile app. The application allows users to search and filter products, add them to the cart, and complete transactions. The app provides a seamless shopping experience across both web and mobile platforms.

## **Features**

- **User Authentication**: Users can sign up, log in, and maintain their session.
- **Product Search & Filters**: Users can search for products and filter them by category, price, and ratings.
- **Shopping Cart**: Users can add products to the cart and view the cartâ€™s contents.
- **Order Management**: Users can place orders and view order history.
- **Responsive Design**: Built with **Angular** for web and **Ionic** for mobile, providing a responsive and smooth shopping experience.
- **Admin Panel**: Admins can manage products, view user orders, and manage product inventory.

## **Technologies Used**

### **Frontend:**
- **Angular**: Framework for building the user interface.
- **Ionic**: Mobile framework for building cross-platform mobile applications (Android and iOS).
- **Bootstrap**: For styling and responsive design.

### **Backend:**
- **Node.js**: JavaScript runtime for the server-side.
- **Express.js**: Web framework for building RESTful APIs.
- **MongoDB**: NoSQL database for storing product, user, and order data.
- **Mongoose**: ODM for MongoDB for easy interaction with the database.

### **Search and Filters:**
- **Product Search**: Users can search for products by name, category, and other attributes.
- **Product Filters**: Filters allow users to narrow down their search results by:
  - **Category**
  - **Price range**
  - **Rating**
  - **Availability**

### **Mobile App:**
- **Ionic**: Used to build a cross-platform mobile app for Android and iOS. The app is fully integrated with the backend, allowing users to browse products, manage their cart, and place orders.
